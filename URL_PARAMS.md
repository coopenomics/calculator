# URL Параметры калькулятора

Калькулятор поддерживает предустановку значений через URL параметры.

## Поддерживаемые параметры

### Основные параметры
- `contribution_amount` или `creator_amount` - Взнос результатом труда (в рублях)
- `invest_amount` или `investor_amount` - Взнос деньгами/инвестиции (в рублях)

### Дополнительные параметры
- `withdrawal_rate` - Процент возврата стоимости (0-100)
- `initial_capital` - Стартовая капитализация (в рублях)
- `membership_fee` - Эффективность капитала (в процентах)
- `monthly_contributions` - Взносы других создателей в месяц (в рублях)
- `growth_multiplier` - Множитель роста взносов (в процентах)
- `membership_fee_growth_multiplier` - Множитель роста членских взносов (в процентах)

## Примеры использования

### Базовый пример
```
https://coopenomics.github.io/calculator/?invest_amount=100000&creator_amount=10000
```

### Расширенный пример
```
https://coopenomics.github.io/calculator/?contribution_amount=50000&investor_amount=200000&withdrawal_rate=80&membership_fee=7
```

### Визард с параметрами
```
https://coopenomics.github.io/calculator/#/wizard?invest_amount=100000&creator_amount=10000
```

## Особенности

1. Все значения должны быть неотрицательными числами
2. Параметры с значением 0 игнорируются
3. Неправильные или некорректные значения игнорируются
4. Параметры работают как для основного калькулятора (/), так и для визарда (/wizard)
5. Поддерживаются алиасы для обратной совместимости:
   - `creator_amount` = `contribution_amount`
   - `invest_amount` = `investor_amount`

## Валидация

- Все значения парсятся как числа с плавающей точкой
- Отрицательные значения игнорируются
- NaN значения игнорируются
- Пустые или отсутствующие параметры используют значения по умолчанию
